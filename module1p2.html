Specifying Layout & Style

Turn a basic structure into a polished design by telling the AI how to arrange and style elements.

The Power of Utility-First CSS

We use **Tailwind CSS**, a "utility-first" framework. Instead of writing custom CSS files, you build designs by applying pre-existing classes directly in your HTML. This is perfect for AI generation because you can describe the styling in plain language.

For example, to create a centered, bold heading with a bottom margin, you would tell the AI to generate this:

<h1 class="text-center font-bold mb-4">My Title</h1>

Controlling Layout with Flexbox & Grid

The two most important tools for layout are Flexbox and Grid. You don't need to be an expert, but knowing what to ask for is key.

Flexbox

Use for arranging items in a **single row or column**. Ask for things like flex, justify-between (to space items out), or items-center (to align them vertically).

Grid

Use for arranging items in a **two-dimensional grid** (rows AND columns). Ask for grid and specify the number of columns, like grid-cols-3 for three columns.

Layout in Action: An Example

Let's ask the AI to create a simple user profile card. First, we'll give it a prompt with no layout instructions, then we'll add them.

The Prompt:

TASK: Create a user profile card.
CONSTRAINTS: It should have a placeholder for an avatar, a name, a username, and a follow button. Use Tailwind CSS.

The Prompt:

TASK: Create a user profile card.
CONSTRAINTS: It should have a placeholder for an avatar, a name, a username, and a follow button. Use Tailwind CSS.
LAYOUT: Use flexbox to align the items. The avatar and the text block (name/username) should be on the left, and the follow button should be on the right. Vertically center all items. The container should have padding and a shadow.
